# Joystick connection procedure 

1. Plug in arduino board to usb port.

2. Connect ROS to Arduino board.

    - $ rosrun rosserial_python serial_node.py /dev/ttyACM0 (*your USB port name)

3. Run ROS joystick filter to listen to joystick data and publish filtered twist velocity message:

    - $ roslaunch joystick_filter joystick_filter.launch
